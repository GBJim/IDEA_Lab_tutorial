  • Python / R / Julia growing at expense of proprietary tools
  • JVM-based Big Data Ecosystem

Distributed in-memory data analysis
  • Apply data operations on arbitrarily large data sets spread across a cluster of machines
  • Programming API and Execution Engine separate problems

Staying Competitive
  • "Enterprise money" is still in the JVM ecosystem
